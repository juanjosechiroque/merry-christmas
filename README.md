# merry-christmas

Merry Christmas! Display a christmas tree and a christmas greeting!


```
      *
     ***
    ****** 
   ********
  **********
 ************
**************
      |

Merry Christmas!
```


## Install

```
npm i juanjosechiroque-merry-christmas
```

## Useage
```js
const merrychristmas = require("juanjosechiroque-merry-christmas")
merrychristmas();
```
